# Security Policy

## Supported Versions

We actively support and provide security updates for the following versions of WP AI Bridge:

| Version | Supported |
| ------- | --------- |
| 1.2.x   | ✅ Yes     |
| < 1.2.0 | ❌ No      |

## Reporting a Vulnerability

If you discover a security vulnerability within WP AI Bridge, please **DO NOT** open a public issue. Instead, report it privately to ensure the security of installations worldwide.

Please send security reports directly via email to: **jean-marie@jmclement.net** (or the email listed in the plugin header).

In your report, please include:
1. **Description of the vulnerability** and its potential impact.
2. **Detailed steps to reproduce** the issue (a Proof of Concept script or curl commands is highly appreciated).
3. **Environment details** (WordPress version, PHP version, other active plugins if relevant).

We will acknowledge receipt of your report within 48 hours and work with you to coordinate a security release to address the issue.

Thank you for helping keep WP AI Bridge and the WordPress community secure!
